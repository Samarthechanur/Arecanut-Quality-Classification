# 🌰 Arecanut Quality Classification

A Deep Learning-based web application that classifies arecanuts into different quality categories using a Convolutional Neural Network (CNN). The application is built with Flask and allows users to upload an image and receive the predicted quality.

## 🚀 Features
- Upload arecanut images
- Predict quality using CNN
- Displays prediction result
- Flask-based web application

## 🛠️ Technologies Used
- Python
- Flask
- TensorFlow / Keras
- HTML
- CSS
- NumPy

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python app.py
```

Open in your browser:

```
http://127.0.0.1:5000
```
